About me

My name is Jenny Pavlova and I am a young and passionate web developer from Sofia, Bulgaria, currently based in Munich, Germany. Most of my experience comes from frontend development with ReactJS, Twitter Bootstrap, HTML5, CSS3. I have also worked with various backend tech stacks including LAMP, MEAN and end-to-end Wordpress solutions.

I mostly tend to work remote from my home/local co-working spaces, however I was a backend developer in a local web development studio, where I learned a lot about collaboration and taking responsibilities.

I have some experience with entrepreneurship - some time ago along with several friends we co-founded YCharge - a startup in the electric vehicle sector. We started off as a Startup Weekend project and won our place in the Start-It-Smart Pre-Accelerator's second batch.

I have around 2 years experience with Salesforce Commerce Cloud ( Demandware ) . As a part of the software development team, I am creating third-party integrations, backend functionalities using JavaScript and Demandware Script, frontend features using JavaScript, jQuery, HTML.

I am focusing on Full-Stack JavaScript development using React, Node.js, Express.js, Docker and MongoDB.
